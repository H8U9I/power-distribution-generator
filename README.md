# 配电箱系统图自动生成程序

可视化管理界面 + 自动负荷计算 + 标准配电箱系统图出图（SVG / DXF / 计算书）。
支持**三相电及三相平衡计算**、**防火等级与线缆类型配置**、**设备/回路层级子菜单**。

---

## 本地软件版（exe，双击即用，推荐）

无需 Python、无需命令行，打包好的**单文件程序**，双击即自动打开浏览器使用。

```bash
# 1. 双击  dist/配电图生成器.exe
# 2. 程序自动启动本地服务并打开浏览器到 http://127.0.0.1:18800
# 3. 正常配置设备/回路，点「导出DWG」即自动调用本机 CAD2014 生成 .dwg
```

- 程序位置：`H:\配电图生成器\dist\配电图生成器.exe`（57MB 单文件，含全部依赖）
- 导出文件位置：默认在 **exe 同目录的 `output\`** 文件夹（如 `配电图生成器/output/配电系统图.dwg`）；若 exe 所在目录不可写，则落到 `文档\配电图生成器\output\`
- 端口：默认 18800，被占用时自动顺延（18801…），浏览器会自动打开实际端口
- 发给别人：直接复制这个 `.exe` 即可。对方电脑也需装 AutoCAD 才能用「导出DWG」，其余功能无需 CAD
- 首次双击会稍等几秒（程序在后台解压），随后浏览器自动弹出

## 网页版启动（备选）

```bash
# 方式一：双击 start_app.bat
# 方式二：命令行
python app.py
# 浏览器打开 http://127.0.0.1:18800
```

启动后左侧配置项目 → 添加设备与回路，右侧实时预览系统图，顶部按钮导出 SVG / DXF / 计算书。

> 依赖：标准库 + `ezdxf`（DXF 导出）+ `openpyxl`（后续 Excel 导入）。本机环境已配好。

## 二、功能对照（需求 → 实现）

| 需求 | 实现 |
|------|------|
| 三相电及三相平衡计算 | 供电系统可选 三相380V / 单相220V；单相回路按电流贪心分配到 L1/L2/L3，输出三相电流与不平衡度 |
| 导出 DWG 格式 | 输出 **DXF**（开放标准，AutoCAD/中望/CAD 直接打开编辑）。DWG 为 Autodesk 私有格式，纯 Python 无法直接写出，在 CAD 中“另存为 .dwg”即可 |
| 内置标准出图样式与回路规则 | 总开关 + 母线 + 分路断路器 + 线缆/空开标注 + RCD 标记，参照常规配电箱系统图 |
| 可视化管理界面，以添加设备为主 | 网页树形结构：配电箱 → 回路类型 → 分路 → 设备；每设备填功率，自动匹配线缆与空开 |
| 防火级别配置 + 线缆类型 | 防火等级（普通/阻燃C/B/耐火/低烟无卤，含降容）、线缆类型（BV/BVR/YJV/NH-BV） |
| 插座回路：总数+分路数，常用/单独分设 | 填常用插座总数与分配路数一键生成；单独电器单独成路 |
| 照明回路：分路数 + 是否分多路 | “分多路”开关 + 照明分路数，一键生成多路照明 |
| 设备回路（单一路设备/组） | 独立添加设备回路，设备列表子菜单逐一添加，支持三相设备 |
| 层级子菜单 | 三类回路均可折叠展开，分路与设备逐项管理，支持高级覆盖（kx/cosφ/RCD/三相） |

## 三、自动匹配规则（简化国标，供参考）

- 单相 220V / 三相 380V，cosφ 默认 0.9，需用系数 kx 按回路类型（插座 0.6 / 照明 0.9 / 设备 1.0）
- 导线载流量：铜芯 BV 穿管暗敷（1.5→16A…25→100A…），防火等级降容（耐火×0.85、阻燃×0.9 等）
- C 型微型断路器，按计算电流向上取标准值；插座/涉水设备强制 RCD 30mA
- 回路空开下限：插座 C16、照明 C10、设备按功率（空调等电机类上浮 C16）
- 最小线径：住宅固定布线 ≥ 2.5mm²；大功率/柜机/热水器 ≥ 4mm²
- 三相平衡：单相回路按电流降序贪心填入当前最小相，给出不平衡度

## 四、目录结构

```
配电图生成器/
├─ app.py            Web 服务（零前端依赖，stdlib http.server）
├─ engine.py         计算引擎：三相平衡 / 线缆选型 / 空开选型 / 防火降容
├─ exporters.py      出图：SVG 预览 + DXF 导出（ezdxf）+ DWG（调 AutoCAD COM）
├─ static/index.html 前端界面（原生 JS，无构建步骤）
├─ start_app.bat     一键启动（网页版）
├─ dist/配电图生成器.exe  本地软件版（双击即用，含全部依赖，57MB 单文件）
├─ make_icon.py      生成程序图标（构建期用）
├─ peidian.py        旧命令行版（保留，peidian.py --demo 仍可跑）
├─ sample_input.*    示例数据
└─ output/           导出产物（配电系统图.svg / .dxf / .dwg / 负荷计算书.md）
```
